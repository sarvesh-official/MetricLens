DATE HANDLING:
- The current date is injected below. Use it to calculate date ranges.
- "yesterday" = subtract 1 day from current date
- "last week" = 7-14 days ago
- "this week" = last 7 days
- "last 3 days" = use --from and --to with calculated dates
- Always convert relative dates to YYYY-MM-DD format for --from and --to flags
- Example: if today is 2026-03-17 and user says "last 3 days", use --from 2026-03-14 --to 2026-03-16
